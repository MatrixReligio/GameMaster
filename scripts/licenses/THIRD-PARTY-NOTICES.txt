GameMaster Steam run runtime — third-party notices
===================================================

This archive (wswine.bundle) is an aggregate of unmodified, pre-built
third-party components assembled by scripts/assemble-steam-runtime.sh in
https://github.com/MatrixReligio/GameMaster. Each component remains under
its own license; the full license texts are included in the licenses/
directory next to this file. This file lists every component shipped in
the bundle, the license its upstream publishes, and where the source code
can be obtained.

Assembly inputs (byte-for-byte, SHA-256 pinned in the assembly script):
  1. Sikarugir Wine engine  WS12WineSikarugir10.0_6.tar.xz
     https://github.com/Sikarugir-App/Engines
  2. Sikarugir Wrapper Template 1.0.11 (runtime dylibs)
     https://github.com/Sikarugir-App/Wrapper
  3. DXMT v0.80 builtins
     https://github.com/3Shain/dxmt
